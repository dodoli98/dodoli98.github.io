ABOUT

안녕하세요, 신입 웹 퍼블리셔 김선규입니다. 사용자 경험 개선을 위해 항상 고민하고 웹 표준과 접근성을 고려하여 모든 사용자가 편리하게 이용할 수 있도록 노력합니다.

특히, 의도와 목적에 맞는 태그 사용의 중요성을 깊이 탐구하고 있으며, 이를 통해 검색 최적화와 사용자 경험 향상에 기여하고자 합니다. 지속적인 성장과 발전을 추구하며, 새로운 도전을 두려워하지 않고 항상 배우는 자세를 가지고 있습니다.

앞으로도 끊임없이 배우고 발전하는 웹 퍼블리셔로서의 역량을 강화해 나가겠습니다. 감사합니다.

  • HTML / CSS

    다양한 사용자들을 배려하여 웹 페이지를 설계하고, 웹 접근성을 고려한 올바른 HTML 태그를 활용할 수 있습니다. CSS를 활용하여 다양한 디자인과 애니메이션을 구성할 수 있고 다양한 디바이스와 화면 크기에 맞춰 반응형 레이아웃을 구성할 수 있습니다.

  • JavaScript / JQuery

    웹 페이지에 최신 트렌드에 부합하는 JavaScript 동적 기능들을 적용할 수 있습니다. 사용자들에게 더욱 흥미로운 웹 경험을 제공하기 위해 끊임없이 노력하고 있습니다. jQuery를 활용하여 DOM 조작 및 이벤트 처리를 간편하게 수행하여 프로젝트에 효율성을 높입니다.

  • React

    React를 활용하여 컴포넌트 기반의 구조적이고 효율적인 웹 애플리케이션을 개발할 수 있습니다. 상태 관리를 통해 더 나은 사용자 경험을 제공할 수 있습니다.

  • Design

    Photoshop을 이용하여 이미지 보정, 합성 및 웹 사이트 디자인과 같은 다양한 실무 디자인 작업을 수행할 수 있습니다.

Toy project

  • How Weather

    React와 OpenWeather API를 이용해 만든 현재 날씨 앱입니다. API요청은 axios를 사용했고 scss를 사용해 스타일 적용을 했습니다. useEffect와 setInterval을 사용하여 페이지에 현재 시간을 실시간으로 표시합니다.

  • 2D Pixel mini Game

    Canvas와 Vanilla JavaScript만을 활용하여 만든 미니 게임입니다. requestAnimationFrame을 통한 부드러운 애니메이션을 구현했습니다. 키보드 이벤트 처리를 통해 실시간 사용자 입력에 반응하며, 랜덤 요소를 활용해 좀 더 다이나믹한 플레이를 만들었습니다. 필요한 기능은 함수로 분리해 만든뒤 main함수에서 무한루프가 되도록 로직을 만들었습니다.

  • Daily Todo

    Vanilla JavaScript로 구현한 할 일 관리 웹 애플리케이션입니다. Local Storage를 활용하여 데이터를 저장하도록 하였습니다. 주요 기능으로는 할 일 추가, 완료 체크, 삭제가 있으며, IME 입력 이슈를 해결하여 다국어를 지원합니다. 중복 항목 방지 기능을 통해 리스트의 일관성을 유지하도록 만들었습니다.
